# Multigame Projet Logiciel
Le projet est une plateforme regroupant plusieurs jeux.
Executez le fichier main.exe pour accéder à la plateforme.

Vous trouverez dans ce dossier :
    -main.exe : executable du projet
    -Rapport_Multigames.pdf : rapport du projet
    -code : fichiers python du projet
    -fonts : polices d'écritures
    -images : images nécessaires pour faire fonctionner le projet
    -lib : librairies nécessaires au main.exe


Requirements :
  -pygame 2.1.2
     =>   installation :
	  pip install pygame
